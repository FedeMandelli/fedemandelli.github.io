# Personal Website

## Development

### SCSS Build Commands
- `npm install`
- `npm run build:css` builds `css/scss/style.scss` into `css/style.min.css`
- `npm run watch:css` watches SCSS changes and rebuilds `css/style.min.css`
- `npm run check:css` compiles once with default Sass output (sanity check)
